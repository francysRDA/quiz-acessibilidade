Guia de Uso do Quiz

  • 🖱️ Com o Mouse: Clique nos botões para selecionar respostas ou avançar. Você também pode clicar e arrastar a barra de rolagem lateral para navegar.
  • 📱 No Smartphone: Toque com o dedo diretamente sobre os botões e alternativas. Use a ação de pinça (dois dedos) para dar zoom na tela e arraste o dedo para rolar a página.
  • ⌨️ Com o Teclado: Use as teclas A e D ou Tab para navegar entre os elementos e Enter ou Espaço para ativar os botões. Use as setas (cima/baixo), Page Up ou Page Down para mexer a barra de rolagem.
  • 🔍 Acessibilidade: Use os botões no topo para aumentar a letra (A+) ou ativar o modo de Alto Contraste se tiver dificuldade de visão.
  • 📖 Aprendizado e Prática: Leia primeiro o conceito teórico e depois clique em "Ir para o desafio" para responder a pergunta.
  • 🔄 Refazer: Se errar, você pode clicar em "Tentar Novamente" para revisar a questão.
  • ⬅️ Voltar: Você pode retornar para reler conceitos ou ver a questão anterior.

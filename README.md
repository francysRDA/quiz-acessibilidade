# Documentação do Recurso Educacional Aberto (REA)

## 1. Descrição do Recurso
- **Título:** Quiz de Acessibilidade e Inclusão Digital.
- **Público-alvo:** Estudantes de tecnologia, desenvolvedores web, educadores e qualquer pessoa interessada em tornar o ambiente digital mais inclusivo.
- **Objetivos Educacionais:** 
  - Compreender os conceitos de acessibilidade e inclusão digital.
  - Identificar barreiras digitais comuns e suas soluções.
  - Conhecer tecnologias assistivas e diretrizes básicas de desenvolvimento acessível (como uso de HTML semântico e contraste).
- **Características de Acessibilidade:** 
  - Compatibilidade com leitores de tela (HTML semântico e ARIA).
  - Modo de Alto Contraste.
  - Redimensionamento dinâmico de fontes.
  - Navegação completa por teclado com foco visível.
  - Design responsivo adaptado para mobile.

## 2. Link para Cópia e Edição
Os arquivos fonte (HTML/CSS/JS puro) podem ser visualizados, copiados e editados livremente. Por ser um arquivo único e autocontido, o próprio `index.html` serve como base para modificações (Open Source / Vanilla JS).

- Implementação da lógica de estado em JavaScript.
